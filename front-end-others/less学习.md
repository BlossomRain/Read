# Less文档学习

- [ ] 笔记时间：2021.06.22

## 1.基本使用

### 1.1 安装

- nodejs 直接使用 npm安装即可
- 浏览器找cdn

### 1.2 使用

- 编译器 lessc 进行处理

- 样式选择器可以嵌套编写，@变量名：变量值 可以声明变量

  &可以实现平级

- 可以直接引用已经声明的样式，可以跟函数一样传入参数，可以设置默认值

- 使用 @import 可以导入其他 less 文件，可以在参数里面放常量用于匹配，放入@_ 代表匹配任何参数，隐式自带参数列表@arguments

- 支持加减运算，继承使用 :extends(.class)进行配置，不可以有参数，继承传入第二个参数 all 可以自动继承所有相关设置

- 避免编译`~"配置"`
